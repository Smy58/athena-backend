# Athena Backend

Бэкенд для клуба **Афина**. NestJS + Prisma + PostgreSQL.
Повторяет всю модель данных, которая раньше жила в localStorage прототипа:
пользователи, персонажи, доска заданий, гильдии, магазин, поддержка, отзывы.

## Что нужно установить

1. **Node.js 18+** — https://nodejs.org
2. **PostgreSQL** — один из вариантов:
   - Docker (проще всего):
     ```bash
     docker run --name athena-db -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=athena -p 5432:5432 -d postgres:16
     ```
   - Бесплатная облачная база — [Neon](https://neon.tech) или [Supabase](https://supabase.com). Тогда локально ничего ставить не надо, просто вставишь connection string в `.env`.

## Установка и запуск

```bash
npm install
cp .env.example .env
# впиши свою DATABASE_URL и любой случайный JWT_SECRET в .env
npx prisma migrate dev --name init
npm run start:dev
```

API поднимется на `http://localhost:3000/api`.

## Структура проекта

```
prisma/schema.prisma       — вся модель данных
src/
  main.ts                  — точка входа
  app.module.ts             — корневой модуль
  prisma/                   — общий Prisma-сервис (доступен везде)
  auth/                      — регистрация, вход, JWT
  users/                     — профиль, гильдия пользователя
  characters/                — персонажи (полный CRUD)
  quests/                    — доска заданий
  guild/                     — контракты гильдий, банка
  shop/                      — звания, зелья, пойки, финики
  support/                   — темы поддержки + чат
  reviews/                   — отзывы на игры и задания
```

## Что уже готово

- Регистрация и вход с хешированием паролей (bcrypt) и JWT-токенами
- Персонажи — полноценный CRUD с проверкой владения
- Задания, гильдии, магазин, поддержка, отзывы — рабочие базовые эндпоинты

## Что стоит доделать дальше
